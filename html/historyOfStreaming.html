What Is Streaming?

Streaming media refers to the simultaneous transmission and playback of video and audio data over a computer network. The process of data transmission itself is called streaming, and transmitted programs are referred to as livestream or short stream.

What is a Containerformat?

In information technology a container format specifies a data structure with which individual data streams of possibly different formats can be merged to form a data stream, for example a container file. There are different possibilities of container formats. The simplest case is to combine several files into one archive file, such as the TAR format. Based on this, the term "container format" is often used in the sense that the data contained in it make a meaningful whole - for example, combining video data with audio data to make a playable movie or combining textual information with images and other information so that they have a composite document. Well-known and often used media container formats, especially for the common storage of the image and sound data of a movie in a single stream, are MP4 (file extension .mp4), Matroska (.mkv, .mka), DivX (.divx) and the outdated AVI (.avi).

Flash Video .flv

Flash Video (FLV) is an open container format developed by Adobe Systems that is primarily used for Internet video content transfer. Depending on the code used, it is compatible with Adobe Flash players version 6 or 7.

MP4

MP4 is a video container format provided by MPEG for MPEG-4 content. The MP4 file format is based on the Apple QuickTime file format. Both formats organize media data in hierarchical and sequential data units. These units are called "atoms," and their TYPE identifiers (moov, trak, mdia, minf, stbl, etc.) are identical. You can record a variety of different media types and save them with structural information and their dependencies. MP4 files can store multimedia content in the form of multiple audio and video tracks as well as subtitles, 2D and 3D graphics. This content can then be played locally using appropriate software or streamed over a network.

MOV

MOV is a file format especially made for the QuickTime Player which means that it works perfectly on Macs. The MOV format was developed by Apple Inc. Later, the largely MP-based MP4 became the international standard. Both, MOV and MP4 use the MPEG 4 codec for compression and are therefore interchangeable in the Apple QuickTime environment. MOV is a widely used format today that is supported by many computers and applications, so that it can also be used to transfer files. Moreover, MOV is suitable for video editing and Internet broadcasting because it can individually customize the primary elements. In MOV files, multiple tracks are stored independently. Thus, the sound, the image quality or the image resolution can be changed individually, which makes the editing very easily.

Advantages: Due to their high compatibility the MOV files can be played on different system environments; further, they only require a small memory space. They can also be used by many different operating systems, including Mac, Windows, iOS and Android.

Disadvantages: MOV uses lossy compression for the video tracks, which means that the video quality is not as high as the original video. In addition, every track (audio, video, text) of the MOV file is encoded with a special codec. An application to play the file must therefore know all these codecs to be able to play them gently and without jerking.
